Saving changes to new or existing roles

The Save button can be used to create a new role or update an existing one.

To create a new role, the following requirements must be met:

  1. Name must be specified.
  2. Business Unit must be specified.
  3. Member's privilege inheritance must be specified.
  4. At least one privilege must be assigned.

To update an existing role, the only requirement is there should be at least one change in one of the General information fields, or in at least one privilege, otherwise the Save operation won't start (a warning message will be shown in the output window).

The Change summary tab can be used to review the changes applied to role privileges before saving them.
