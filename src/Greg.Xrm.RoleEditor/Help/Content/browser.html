Role Browser

The role browser provides a visual representation of the roles in the system, organized within the Business Unit hierarchy as modeled in your Dataverse environment. You can load the roles for the current environment by clicking the "Load tables, privileges, and roles" button in the main window.

Roles will be displayed in a tree structure, where the root node represents the organization, the child nodes represent the business units, and the leaf nodes are the actual roles. You can change the XrmToolbox connection while the tool is open and, by clicking the same button, load the roles of the new environment, which will be displayed under a new root node. This feature is particularly useful if you want to compare roles between different environments.

Main Features

You can use the Search bar to filter roles by name. The search is case-insensitive and will match any role containing the search term.
By default, only customizable roles are displayed. You can show all roles by clicking on the Hide not customizable roles toggle menu.
Non customizable roles are displayed in gray and are read-only.

If you want to show/hide roles belonging to managed solutions, you can do so by clicking on the Hide managed roles toggle menu.

You can change the default values for the Hide non customizable roles and Hide managed roles toggles using the Settings dialog.

Environment and business unit nodes of the tree can be expanded and collapsed by clicking on the plus and minus icons.

How to create new or edit existing roles

From the role browser, you can create new roles by clicking on the New role button. That will open the Role Editor window, where you can configure the new role data and privileges.

You can also right click on a given role to open a context menu with the following options:

  • Clone current role: available only if just one role is selected.
  • Edit multiple roles: available only if 2 to 10 roles belonging to the same environment are selected. This command opens the Bulk Role Editor window.

Double-clicking an existing role will open the Role Editor window, where you can view and edit the role configuration (if the role is customizable).

Role Inspector

By right-clicking on a given role you can also launch the Role Inspector, a feature designed to help you assess where a given role is actually used.

Role Comparer

If you select 2 roles (even from different environments), and then right-click on one of them, you can launch another hidden gem: the Role Comparer. It compares the privileges assigned to the two roles and shows you the differences.
