Summary

_n.RoleEditor provides an advanced, user friendly, UI to manipulate dataverse security roles.
His main purpose is to overcome the limitations of the new security role editor available via Power Apps Maker Portal.

With this tool you can:

  • Open more than one role at time, even from different environments.
  • Create new roles from scratch, or copying the minimum privileges from existing roles.
  • Modify existing roles.
  • Set privilege depth privilege by privilege, or massively (by table, by operation, or applying specific filters)
  • Use CTRL+C / CTRL+V to copy/paste privilege depth configurations between tables or between single miscellaneous privileges.
  • Export a role definition in excel or markdown.

Steps

  1. Connect to the environment that contains the roles to edit.
  2. Click on Load tables, privileges and roles to fetch the list of roles from the environment.
  3. Here you can
    • Use the Search box to filter the list of roles
    • Use the Filter toolbar button on the right of the Search box to show/hide uncustomizable or managed roles
    • Double click on a given role to enter edit mode.
    • Use the Create new role toolbar button to create a new role.
