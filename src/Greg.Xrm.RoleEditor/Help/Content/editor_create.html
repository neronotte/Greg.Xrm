Create a new role

From the role browser, you can create new roles by clicking on the Create new role button. This opens the Role Editor window, where you can configure the new role.

The tool provides 4 different ways to create a new role:

  1. Start from an empty role: This option creates a new role with no privileges or access levels. If you have more than one environment loaded, the new role will be created in the environment of the currently selected role. If no role is selected, the first environment in the list is used.
  2. Clone selected role: This option creates a new role with the same privileges and access levels as the selected role. The new role will be created in the environment of the currently selected role.
  3. From "Basic User": This option creates a new role with the same privileges and access levels as the "Basic User" role (if present in the selected environment). The new role will be created in the environment of the currently selected role.
  4. From "App Opener": This option creates a new role with the same privileges and access levels as the "App Opener" role (if present in the selected environment). The new role will be created in the environment of the currently selected role.

Once selected the creation method, a new Role Browser window will spawn, set onto the General information tab where you can specify the name, description, parent Business Unit and member's privilege inheritance of the new role.

All info except description are required to save the role.
