Editor

_n.RoleEditor offers an advanced, user friendly, UI to manipulate dataverse security roles.
His main purpose is to overcome the limitations of the new security role editor available via Power Apps Maker Portal, providing a familiar and intuitive configuration experience with a wealth of useful features.

  • Saving changes to new or existing roles
  • Add role to solution
  • Export in Excel or Markdown
  • Import from Excel
  • Copy/Paste all privileges
