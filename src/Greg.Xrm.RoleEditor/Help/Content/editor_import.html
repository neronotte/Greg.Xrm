Import from Excel

File exported via the Export in Excel feature can be updated and re-imported back into the Role Editor. An exported file looks like this:

The only editable cells are the ones containing the privilege levels, and the possible values are automatically validated against the ammissible levels for each specific privilege.

To import a role configuration from an Excel file, click on the Import button on the Role Editor toolbar:

Select the Excel file containing the role configuration to import, and click on the Open button. The system will initially check if the file is valid. Only files that have been actually exported from _n.RoleEditor can be re-imported.

After the file is validated, the system will display a confirmation prompt to inform the user that any unsaved change will be overwritten.

If the user confirms the operation, the system will import the role configuration from the Excel file, and the Role Editor will be updated accordingly.
