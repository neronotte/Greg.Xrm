Copy all privileges

The Copy all privileges button creates a text-based snapshot of the current role, including all its privileges and their access levels. This snapshot is stored in memory and can be pasted to another role using the Paste all privileges button.

This feature is particularly useful when you want to duplicate or merge privileges from one role to another, ensuring consistency and saving time.

Paste all privileges

The Paste all privileges button applies a previously copied privilege snapshot to the currently selected role. This action will merge the privileges from the snapshot with those in the target role using "additive" rules:

  • If a privilege from the snapshot has a higher access level than the current role, the higher level is applied
  • If a privilege from the snapshot has a lower or equal access level, the current role's level is preserved
  • Unknown tables or privileges in the snapshot are skipped without causing errors

This ensures that no privilege is downgraded during the paste operation, maintaining the highest level of access for each privilege.

Validation and Feedback

The paste operation includes comprehensive validation and provides detailed feedback:

  • Success messages show how many privileges were applied and skipped
  • Error messages identify specific format or validation issues
  • Warning messages alert you to empty clipboard or unknown privileges
