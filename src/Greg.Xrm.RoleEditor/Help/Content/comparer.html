Role Comparer

The Role Comparer is a tool designed to show the differences between two roles in a quick and effective way. It's really useful the design phase of your security model, but also when troubleshooting security-related issues.

The Role Comparer can be launched from the Role Browser, by selecting 2 different roles (even from different environments), and then right-click > Compare roles...

A window like the following one will pop up in the main area of the plugin:

The comparison results are show in a tree grid layout, with the following columns:

  1. Group / Privilege Name: name of the privilege or the privilege group for which there is a difference
  2. Role 1: shows the value of the privilege on the first role
  3. Role 2: shows the value of the privilege on the second role

To simplify the reading, privileges are grouped in the following way:

  • Table-Related privileges
    • Table name
      • Privilege name
  • Miscellaneous privileges
    • Privilege name
